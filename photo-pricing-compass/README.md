# Photo Pricing Compass

A strategic pricing calculator for headshot and branding photographers.

## Features
- **CODB Calculator:** Determine your Cost of Doing Business.
- **Scenario Builder:** Create quotes for Individuals, Teams, and Retainers.
- **AI Negotiation:** Get advice on how to handle budget objections.
- **Licensing:** Calculate usage fees based on territory and duration.
- **History:** Save and retrieve past quotes.

## Setup
1. Clone repository
2. Run `npm install`
3. Create a `.env` file with `API_KEY=your_google_gemini_key`
4. Run `npm run dev`
